# PPMpy

A simple gamma-law gas PPM hydrodynamics solver, written with
visualization of the steps in mind.

## Installation

From PyPI:

```
pip install ppmpy
```

From source:

```
pip install .
```

## Running

See the documentation at: https://python-hydro.github.io/ppmpy/intro.html
for usage examples.
